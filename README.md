
# BIME WEBSITE



## Tech Stack

**Frontend:** Nextjs
**Styling:** Tailwindcss
**Deployment:** Vercel


## Installation & Setup

1.Clone the project
```bash
  git clone https://github.com/Omo-Akeye/Revamp.git
  cd Revamp
```
2.Install dependencies

```bash
  npm install
```
3.Install dependencies

```bash
  npm install
```

4.Start the development server

```bash
  npm run dev
```
